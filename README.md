# Forelesning 19 - JDBC Optional, auto-increment, transactions

> Forelesningskoden finner du i `src/main/java/lecture`-mappen.
> 
> SQL-koden finner du i db-skript `files/brettspill.sql`-filen. 
> Kopier koden og kjør en i MySql workbench.
> 
> 
> Sett opp selv `files/brettspill.properties`-filen med
> 
> host=localhost
> db_name=brettspill
> port=3306
> uname=<uname> //ditt brukernavn
> pwd=<pwd>  // ditt passord
> 
> merk at filen står i `.gitignore` for å unngå at passord havner i GitHub.
